从分析到执行的完整流程
▼
1
早上开盘前：基本面打分（10分钟）查CBOT美豆+天气+到港 → 今日偏多/偏空/中性
2
开盘前：技术面打分（5分钟）看日线K线 → 均线/MACD/支撑压力/趋势线 → 偏多/偏空/中性
3
方向确认：两者必须一致才操作都偏多→做多 / 都偏空→做空 / 矛盾→不操作
4
盘中：等待精确入场信号等价格到支撑/压力位 → 出现K线形态 → 等下一根K线确认
5
开仓前：入场清单≥8项才开仓计算盈亏比≥1.5:1 → 确定手数 → 止损止盈设好
6
持仓期间：只有两种处理顺利运行→不动让系统跑 / 反向运行→严格止损
7
过夜前：5个额外问题今晚有无重大数据？美豆方向？基本面支持？持仓量变化？止损够宽？
8
收盘后：写交易日志开仓理由 / 盈亏 / 做对了什么 / 做错了什么 / 下次改进
四种方向组合判断
⚔️
技术面 × 基本面双维度组合
▼
技术面偏空↘基本面偏空↘→最强做空，积极入场 🟢🟢
技术面偏多↗基本面偏多↗→最强做多，积极入场 🔴🔴
技术面偏空↘基本面偏多↗→信号矛盾，轻仓或不操作 ⚠️
技术面偏多↗基本面偏空↘→可能只是技术反弹，快进快出 ⚠️
🚫 6条铁律 — 永远不能做
  • 止损触发后手动取消等回来 → 爆仓根本原因
  • 没有信号就扩大止损距离 → 小亏变大亏
  • 价格大涨/大跌后才追入 → 追涨杀跌必亏
  • 连续止损后立即加大仓位报复 → 情绪交易最危险
  • 技术面和基本面方向矛盾时强行交易 → 胜率不足50%
  • 每天频繁交易超过3次 → 手续费磨光账户
