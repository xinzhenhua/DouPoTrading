⭐ 重要交易口诀（每天开盘前必读）
📊 MACD核心口诀（最重要！）
1️⃣
0轴上方金叉 = 趋势做多，最强信号，DIFF和DEA都在0轴上方时出现金叉，代表主趋势偏多
⚠️
0轴下方金叉 = 空头反弹，不是趋势反转！只做短线，不追涨，随时准备反手做空。典型案例：2887→2986这波反弹就是0轴下金叉
2️⃣
0轴下方死叉 = 趋势做空，最强信号，DIFF和DEA都在0轴下方时出现死叉，做空信心最强
⚠️
0轴上方死叉 = 多头回调，趋势不变，不要急着做空，只是短暂回调，主趋势仍偏多
3️⃣
顶背离 = 反弹结束信号：价格创新高，但DIFF没有创新高 → 上涨动能衰竭 → 做空机会来临
4️⃣
底背离 = 下跌结束信号：价格创新低，但DIFF没有创新低 → 下跌动能衰竭 → 做多机会来临
5️⃣
STICK柱子变短 = 动能减弱，注意方向改变；两线接近 ≠ 立刻反转，必须等金叉或死叉出现才入场
6️⃣
DIFF与DEA差距扩大 = 趋势动能增强，顺势持仓；差距收窄 = 动能减弱，考虑减仓或等待出场
📈 均线系统核心口诀
1️⃣
空头趋势中，均线是天花板（做空的最佳卖出位）：价格反弹到MA5/MA10/MA20 → 出现K线信号 → 做空
2️⃣
多头趋势中，均线是地板（做多的最佳买入位）：价格回调到MA5/MA10/MA20 → 出现K线信号 → 做多
3️⃣
均线压缩警报：MA5开始向MA10靠近，两线差距缩小 = 动能减弱 = 趋势可能改变，提高警惕
4️⃣
价格突破MA40/MA60，但MA5还在MA40下方 = 强反弹不是趋势反转，谨慎追涨
5️⃣
均线完美空头排列（MA5<MA10<MA20<MA40<MA60）+ 均线全部向下 = 最强做空环境
6️⃣
MA5上穿MA10金叉需配合MACD金叉确认，单独均线金叉在震荡市假信号很多
🕯️ K线与价格行为口诀
1️⃣
高位长上影线 = 有人在卖，短期遇阻：上影线越长，压力越强，是做空的重要前期信号
2️⃣
低位长下影线 = 有人在买，短期获支撑：下影线越长，支撑越强，是做多的重要前期信号
3️⃣
K线形态必须在关键位置出现才有效（支撑位/压力位/均线/趋势线），随机位置出现完全忽略
4️⃣
看到K线形态后不能立刻入场，必须等下一根K线收线确认（阳线确认做多，阴线确认做空）
5️⃣
连续多根阳线后出现小实体 = 多方力竭；连续多根阴线后出现小实体 = 空方力竭，双方都需等K线确认
📦 量价关系口诀
✅
价涨量增 = 健康上涨，多方真实进场，趋势可能延续，可以跟多
⚠️
价涨量缩 = 反弹末期，动能不足，多方力量减弱，警惕反转做空
✅
价跌量增 = 加速下跌，空方真实进场，趋势可能延续，可以跟空
⚠️
价跌量缩 = 下跌无力，空方力量减弱，警惕止跌反弹
🔍
减仓收阳（日增仓为负）= 空头回补推动，不是真实多头买盘，反弹不可持续，后续大概率继续下跌
🔍
增仓收阳（日增仓为正）= 真实多头进场，趋势可能转变，需配合均线和MACD确认
🧱 支撑压力口诀
1️⃣
下跌趋势中：均线 = 压力（卖出位）；上涨趋势中：均线 = 支撑（买入位）
2️⃣
支撑被有效跌破后反弹到此 = 变成压力，做空机会；压力被有效突破后回踩 = 变成支撑，做多机会
3️⃣
一个价位被触碰次数越多，它越重要；整数关口（2900/3000）往往是天然强支撑/压力
4️⃣
有效突破必须收盘确认 + 成交量放大，仅日内高低点触碰不算真正突破
⏰ 多周期使用口诀
📅
日线 = 定方向（每天早上看一次）：判断今天应该做多还是做空，找关键支撑压力位
⏱️
1小时线 = 找入场（价格到位时切换看）：等待K线形态出现，等确认K线，精准入场
📉
分时图 = 监控持仓（入场后用）：看白线和均价线关系，判断当日多空力量
⚠️
日线和1小时线方向必须一致才入场；方向矛盾时不操作，等待一致再说
🎯
止盈距离决定持仓时间：30点内=日内；50-100点=2-4天；100点以上=1-2周

1
第一步 · 3分钟
查CBOT美豆昨晚收盘
文华随身行搜"美豆主连"查日线
涨超5点 → 今日偏多倾向
跌超5点 → 今日偏空倾向
涨跌在5点以内 → 影响不大，看技术面
2
第二步 · 5分钟
看豆粕日线K线图
均线排列判断：
MA5<MA10<MA20 且向下 → 空头趋势，优先做空
MA5>MA10>MA20 且向上 → 多头趋势，优先做多
均线纠缠/走平 → 震荡行情，不操作

MACD判断：
DIFF和DEA都在0轴下方+死叉 → 最强做空信号
0轴下方出现金叉 → 只是反弹，不是反转！
DIFF和DEA都在0轴上方+金叉 → 最强做多信号

趋势线判断：
价格在下降趋势线下方 → 偏空
价格突破趋势线收盘站稳 → 考虑偏多
📌 得出今日方向：偏多 / 偏空 / 观望（不明确就选观望）
3
第三步 · 3分钟
确定今日关键价格
从日线图找出今日关键价位，记录在纸上或交易笔记：
上方第一压力：___（均线位 or 历史高点）
下方第一支撑：___（均线位 or 历史低点）
趋势线当日位置：___

今日计划：等价格到___附近，看1小时线找做___的信号
4
第四步 · 2分钟
完成打分器+基本面确认
打开工具→打分器Tab，完成今日打分
基本面和技术面方向一致？
都偏多 → 可以积极找做多机会
都偏空 → 可以积极找做空机会
方向矛盾 → 今日不操作，等待一致

今天有没有重大数据发布？（每周四深夜USDA，每月10日WASDE）
有重大数据 → 收盘前平仓，不持隔夜
🎯
盘中等待入场信号（不需要一直盯）
当价格接近关键位时才切1小时图

5
第五步 · 触发条件
什么时候切换到1小时图？
不需要一直盯着1小时图！
只有当价格靠近关键位时才看：

做空场景：价格反弹到均线压力/趋势线附近（±10点）
做多场景：价格回调到均线支撑/历史低点附近（±10点）

设置文华随身行价格提醒：
→ 到达目标区域时手机震动提醒
→ 收到提醒再打开看1小时图
6
第六步 · 5分钟
看1小时线判断信号强度
看均线：1小时均线方向和日线方向一致吗？
一致 → 信号更可靠
相反 → 谨慎，可能是短期波动

看MACD背离（最重要！）：
做空时：价格创新高但1小时DIFF没创新高 → 顶背离！做空信号强
做多时：价格创新低但1小时DIFF没创新低 → 底背离！做多信号强

看K线形态：
出现射击之星/看跌吞噬/十字星 → 做空前期信号
出现锤子线/看涨吞噬/早晨之星 → 做多前期信号

看上影线/下影线：
长上影线 = 高位有卖压，辅助做空信号
长下影线 = 低位有买盘，辅助做多信号
📌 信号强弱判断：背离+形态+影线 = 强信号；单独出现一个 = 弱信号，等等看
7
第七步 · 关键步骤
等1小时K线确认收线
这是最多人犯错的步骤：看到信号就立刻入场！

做空：必须等当根1小时K线收盘为阴线后才入场
做多：必须等当根1小时K线收盘为阳线后才入场

收线之前K线还会变化，你看到的形态可能消失
宁可错过，不可冒进！

1小时K线何时收线？
→ 09:00、10:00、11:00、13:30、14:30...整点收线
→ 夜盘：21:00、22:00、23:00收线
8
第八步 · 入场前2分钟
入场清单最终确认
快速核查以下7项（全部满足才开仓）：
□ 日线方向确认（偏多/偏空）
□ 1小时方向一致（不矛盾）
□ 出现有效K线形态并已收线确认
□ 盈亏比 ≥ 1.5:1（止盈距离 ÷ 止损距离）
□ 止损位置有技术依据（不是随意设的）
□ 手数 ≤ 3手，风险 ≤ 账户2%
□ 今天没有重大数据（或不持隔夜）

7项全满足 → 开仓
有任何一项不满足 → 继续等待
⚡
下单执行（2分钟完成）
开仓后立刻设止损止盈，不能拖！
▼
9
第九步 · 立刻执行
下单 + 立刻设止损止盈
下单选择限价单，价格设在信号K线收盘价附近

做空时止损设置逻辑：
止损 = 入场压力位上方10-20点（如果到这里说明判断错了）
示例：压力位2990，入场2985，止损3005

做多时止损设置逻辑：
止损 = 入场支撑位下方10-20点
示例：支撑位2940，入场2945，止损2925

止盈设置：设在下一个明显支撑/压力位附近
第一目标止盈：较近的位置（主仓位2/3在此止盈）
第二目标止盈：较远的位置（剩余1/3继续持有）
🚨 止损设好之前不能离开手机！这是最重要的一步
📊
持仓期间管理（每天只看2次）
不要频繁看，每天早晚各一次即可

四种方向组合判断
⚔️
技术面 × 基本面双维度组合
▼
技术面偏空↘基本面偏空↘→最强做空，积极入场 🟢🟢
技术面偏多↗基本面偏多↗→最强做多，积极入场 🔴🔴
技术面偏空↘基本面偏多↗→信号矛盾，轻仓或不操作 ⚠️
技术面偏多↗基本面偏空↘→可能只是技术反弹，快进快出 ⚠️
🚫 6条铁律 — 永远不能做
  • 止损触发后手动取消等回来 → 爆仓根本原因
  • 没有信号就扩大止损距离 → 小亏变大亏
  • 价格大涨/大跌后才追入 → 追涨杀跌必亏
  • 连续止损后立即加大仓位报复 → 情绪交易最危险
  • 技术面和基本面方向矛盾时强行交易 → 胜率不足50%
  • 每天频繁交易超过3次 → 手续费磨光账户
