🚫 6条铁律 — 永远不能做
  • 止损被触发后手动取消，等价格回来 → 爆仓的根本原因
  • 没有信号就扩大止损距离 → 让小亏变成大亏
  • 价格大涨/大跌后才追入 → 追涨杀跌必亏
  • 连续止损后立即加大仓位报复 → 情绪交易最危险
  • 技术面和基本面方向矛盾时强行交易 → 胜率不足50%
  • 每天频繁交易超过3次 → 手续费磨光账户

完整交易决策流程（8步）
🔄
从分析到执行的完整闭环
每次交易都必须走完这8步
▼
1
早上开盘前：基本面打分（10分钟） 查CBOT美豆收盘 + 天气 + 到港预报 → 基本面今日偏多/偏空/中性
│
2
开盘前：技术面打分（5分钟） 看日线K线 → 均线/MACD/支撑压力/趋势线 → 技术面偏多/偏空/中性
│
3
方向确认：两者必须一致才操作 基本面+技术面都偏多→做多 / 都偏空→做空 / 矛盾→不操作
│
4
盘中：等待精确入场信号 等价格到支撑/压力位 → 出现K线形态 → 等下一根确认
│
5
开仓前：10项清单（≥8项才开） 计算盈亏比≥1.5:1 → 确定手数 → 止损止盈位置确定
│
6
持仓期间：只有两种处理 顺利运行：不动，让系统跑到止盈 / 反向运行：严格止损，不扛亏损
│
7
过夜前：5个额外问题 今晚有无重大数据？美豆方向？基本面支持过夜？持仓量变化？止损够宽？
│
8
收盘后：写交易日志 记录：开仓理由 / 盈亏结果 / 做对了什么 / 做错了什么 / 下次改进
